# CAP AI Agent – Crypto Agent Protocol compatible

A simple, powerful AI agent that complies with the **Crypto Agent Protocol (CAP)**.  
It receives jobs via an HTTP API, verifies payment on the Sepolia testnet, executes the task, and returns a signed response.

## Features

- ✅ CAP‑compatible (coordination, settlement, signing)
- ✅ Runs locally (completely free)
- ✅ Verifies payment on testnet (no real money)
- ✅ Easy to customise the task (news summariser, price analysis, automation)
- ✅ Clean, commented, MIT‑licensed code

## Requirements

- Python 3.10 or later
- A free [Infura](https://infura.io) account (to get a Sepolia RPC URL)
- A Sepolia testnet wallet (e.g. MetaMask)
- Free testnet ETH from a Sepolia faucet
